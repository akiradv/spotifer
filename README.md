# Spotifer 🎵

Um projetinho pessoal pra eu organizar minhas músicas, ver a duração e descobrir se é single, EP ou álbum.

Depois de 6 meses eu voltei aqui e atualizei, e pretendo atualizar mais.
Agora na **V2** tem um tema claro/escuro e mais firulas (porque eu merecia).

## O que dá pra fazer agora:
+ Arrastar músicas ou selecionar pelos arquivos 
- Ver a duração total e classificação automática (Single / EP / Álbum)
+ Reordenar as faixas arrastando
+ Renomear as músicas (clique duplo no nome)
- Botões de ordenar por nome ou duração
+ Tema claro e escuro (com botão de configurações)
- O site lembra suas músicas mesmo se fechar (nos navegadores que suportam)

## Tecnologias
- HTML, CSS e JavaScript
+ Font Awesome pros ícones
+ IndexedDB e localStorage pra guardar as coisas

## Melhorias futuras

- Player de música integrado
- Exportar página do álbum em um site HTML, bem bonitinho
